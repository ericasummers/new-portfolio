Hangman Wars

Languages/Tools used:

PHP, mySQL, CSS, HTML, Composer, Silex, Twig, JavaScript, jQuery

Github: Project Link

Description: This web application is a game we created as a team of four for an Epicodus school project. The game titled 'Hangman Wars' allows players to travel across a map of authors, trying to guess famous quotes before their army runs out of soldiers. Each successful phrase unlocked gives the player more soldiers and unlocks the next level on the map until they have gotten all the way across to victory. This game was an interesting exercise in creating classes and storing and parsing phrases in PHP and then passing them across to JavaScript as JSON files that were then used on the front end for the gameplay portion.
