PLAY | Experiments, personal projects, and various ideas I'm working on

Color Clicker

Languages/Tools used:

HTML, CSS, Javascript, jQuery, PHP, mySQL

Github: Project Link

Description: This website features a game called 'Color Clicker', created by myself and Dylan Stackhouse, as a fun exercise to practice initially with JavaScript and timing events. Once the basic functionality was created allowing a user to click on different colored boxes and score different amounts of points, a database was added to keep score, and the project was converted to canvas. This project is still ongoing.


Avast Webcomic

Languages/Tools used:

PHP, mySQL, CSS, HTML

Live site: Avast Webcomic

Description: This website is for the online webcomic called "Avast", the story of a young girl who dreams of being a pirate, and her pet catasaur. The comic is not currently being updated but makes use of PHP and a mySQL database to store and display the existing comic pages. All artwork and graphics are my original works.


Terpsichore Lyra

Languages/Tools used:

HTML, CSS, jQuery, Javascript

Live site: Terpsichore Lyra

Description: This is a personal website created in Summer 2014 for the aerial acrobatic duet called Terpsichore Lyra (of which I am a member!). The website has basic contact information and media to help advertise the acrobatic duet and get contacts for performances.
